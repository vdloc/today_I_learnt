- We can specify an start index for an ordered list by

  ```html
  <ol start="4">
    <li>One</li>
    <li>Two</li>	
    <li>Three</li>	
  </ol>
  ```

  and it will be rendered as

  4.One

  5.Two

  6.Three

  We can reverse the order by

  ```html
  <ol start="4" reversed>
    <li>One</li>
    <li>Two</li>	
    <li>Three</li>	
  </ol>
  ```

  and the result is

  4.One

  3.Two

  2.Three
